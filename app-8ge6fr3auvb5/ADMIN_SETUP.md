## Admin Capabilities

As an admin, you have access to:

- **User Management**: View all registered users
- **Role Assignment**: Promote users to admin or demote them to regular users
- **System Overview**: Monitor user activity and system statistics
- **Full Access**: Access to all features and pages in the application

## Security Notes

- The password is securely hashed using Supabase Auth
- Only users with the 'admin' role can access the admin panel
- Regular users cannot see or access admin features

## First Time Login

If this is your first time logging in:

1. Navigate to the application URL
2. You'll be redirected to the login page
3. Enter the admin credentials provided above
4. After successful login, you'll be taken to the dashboard
5. Your session will be saved automatically - you won't need to sign in again unless you explicitly sign out

## Changing Admin Password

To change the admin password:

1. Sign in with the current credentials
2. Go to "Settings" from the sidebar
3. Update your password in the account settings
4. Save changes

## Adding More Admins

To promote other users to admin:

1. Sign in as admin
2. Go to "Admin Panel"
3. Find the user you want to promote
4. Click the "Make Admin" button next to their name
5. The user will immediately have admin access

## Support

If you encounter any issues with the admin account:

1. Check that you're using the correct email and password
2. Ensure your browser allows cookies (required for session persistence)
3. Try clearing your browser cache and cookies
4. Check the browser console (F12) for any error messages
5. Refer to TROUBLESHOOTING.md for common issues and solutions
